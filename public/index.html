    Available Endpoints

    1. HTTP Streaming Endpoint

    URL: POST /eb_stream

    Description: Send prompts and receive streaming responses via HTTP.

    Example usage:

    fetch('/eb_stream', {
    method: 'POST',
    headers: {
        'Content-Type': 'application/json'
    },
    body: JSON.stringify({
        prompt: 'Your prompt here'
    })
});

    2. WebSocket Streaming Endpoint

    URL: WebSocket /ws_stream

    Description: Connect and receive streaming responses via WebSocket.

    Example usage:

    const ws = new WebSocket('ws://' + window.location.host + '/ws_stream');
ws.onopen = () => {
    ws.send(JSON.stringify({
        prompt: 'Your prompt here'
    }));
};
ws.onmessage = (event) => {
    console.log('Received:', event.data);
};
